# Deploy parse-server and parse-dashboard in 5 minutes

Steps: 
- Make a ubuntu 16.04 VPS instance with your favorite provider
- login as root
- copy script to /root
- `chmod +x deploy-parse.sh`
- `./deploy-parse.sh`

And in several minutes you're done!
